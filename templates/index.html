What is Sansans To Talk ?

Sansans To Talk is a web application with which you can create events about series and movies to talk and discuss with people with the same tastes as you. The meetings are developed through meetings by Google Meet, which will be created and shared by the creators of the events.


How it works Sansans To Talk ?

So that you can create your events and talk with people, you must first register, after this you will have two new options: "Event List", where you can see the events that have been created by people, and "Create Event", section where you can create your own events. It is important that after creating the event you go to the "details" section and click on "Start Meeting", this will create a Google Meetings session. The next step you must do is copy the session invitation link and paste it in the "details" section so that people can join.
